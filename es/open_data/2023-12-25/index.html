GitHub asegura el futuro de los datos públicos al moverlos a una bóveda en Noruega

Published: 2023-12-25

GitHub, una plataforma de Microsoft, ha hecho algo importante para guardar y hacer accesibles los datos públicos para el futuro. Han movido sus archivos a una bóveda en Noruega, en una mina de carbón antigua en Svalbard. Esta bóveda tiene mucha información, con un tamaño total de 21 terabytes.

¿Qué ha hecho GitHub para guardar los datos públicos y hacerlos accesibles para el futuro? GitHub movió sus archivos a una bóveda en Noruega, en una mina de carbón antigua en Svalbard. Esta bóveda tiene mucha información, con un tamaño total de 21 terabytes.

¿Cuál es el objetivo principal de esta acción? El objetivo principal de esta acción es proteger los datos en caso de un desastre global y asegurarse de que se puedan leer en el futuro.

¿Cómo han protegido los datos digitales en caso de que los medios digitales se vuelvan viejos? Para hacer esto, han impreso los datos digitales en 186 películas con códigos QR. Así, incluso si los medios digitales se vuelven viejos, se pueden recuperar y usar los datos.

¿Qué tipo de datos se guardan en esta bóveda en Noruega? Además de los archivos públicos de GitHub, también se guardan software de código abierto y datos de instituciones importantes como la Agencia Espacial Europea, la Biblioteca del Vaticano y otros museos y archivos.

¿Qué muestra esta acción sobre el compromiso de GitHub? Esta acción muestra que GitHub se compromete con la transparencia y el acceso a la información.

¿Por qué es importante el concepto de datos abiertos en un gobierno abierto y transparente? El concepto de datos abiertos es importante en un gobierno abierto y transparente porque los datos públicos, compartidos y liberados, son una herramienta poderosa para tener transparencia y responsabilidad.

¿Qué se promueve al hacer que los datos sean accesibles? Al hacer que los datos sean accesibles, se promueve la participación de la gente y se fortalece la democracia.

¿Qué impacto tiene la acción de GitHub en la preservación de los datos y la promoción de la transparencia? La acción de GitHub es un paso importante para preservar los datos y promover la transparencia. Al asegurarse de que los datos estén disponibles para el futuro, se está construyendo un legado de información que se puede usar para tomar decisiones informadas y mejorar la sociedad en general.

En resumen, mover los archivos públicos de GitHub a una bóveda en Noruega es un buen ejemplo de cómo se están protegiendo y guardando los datos públicos. Esta acción promueve la transparencia, el acceso a la información y el gobierno abierto. Al hacer que los datos sean accesibles, se está construyendo un futuro en el que todos puedan tener acceso a la información y usarla para crear un mundo mejor.
