CERN : Large Hadron Collider Beauty experiment releases first set of data to the public

The worlds largest particle physics experiment, the Large Hadron Collider Beauty, is now accessible to anyone in the world, according to the CERN open data portal. Scientists have been using the data to produce their results for public publishing.
