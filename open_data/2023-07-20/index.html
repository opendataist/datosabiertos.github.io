Confluent Launches 'Connect with Confluent' Partner Program for Real-Time Data Streams

Published: 2023-07-20

Data streaming company Confluent has launched a partner program called ‘Connect with Confluent’ to help technology providers enable faster access to real-time data streams. The program allows vendors to bring fully managed data streams directly to their customers through native integrations with Confluent Cloud. This initiative aims to unlock new experiences for end users and create a network effect that opens avenues for growth and innovation. Several technology vendors, including Google Cloud, Elastic, and MongoDB, have already joined the program.

A recent study found that rates of teen suicide and suicide attempts were lowest during the pandemic when schools were shut down. The number of incidents increased between 2016 and 2019 but decreased when schools closed in 2020. However, when classes resumed in 2021, the rate not only rebounded but exceeded expectations. The researchers suggest that interventions during peak months could help protect against seasonal increases in teen suicidality.

Code for Pittsburgh, a meetup group for software engineers, is using their tech skills to make Pennsylvania state legislation more accessible. They are creating a searchable database that tags pieces of legislation and using language models to summarize the contents for easier understanding. The group also plans to set up a notification system for residents to receive alerts when relevant laws are introduced. Code for Pittsburgh is a community where technologists can meet and give back while forging new connections.

The Research Data Alliance Europe (RDA) and the European Open Science Cloud (EOSC-Future) have joined forces to facilitate access to research data. RDA serves as a platform for discussion and knowledge exchange, while EOSC-Future focuses on developing tools and resources. The two entities are working together on the EOSC Future project, which aims to provide European researchers with access to data that adhere to principles of findability, accessibility, interoperability, and reusability (FAIR). The collaboration between EOSC and RDA enhances data interoperability and enables cross-domain research.
