Real - time data project launched by Oxfordshire County Council to tackle traffic congestion

Two researchers have completed a project to ensure Oxfordshire’s transport systems have access to the same data on which to make real-time decisions for the transport network using shared data. They say they are looking forward to creating smarter infrastructure.
