Linux Foundation Announces Overture Maps Foundation to Build Interoperable Open Map Data

The Linux Foundation has formed a joint effort to develop interoperable open map data to strengthen mapping services worldwide, according to the latest announcement from the company's chief executive, the Overture Maps Foundation (Overture). Changes will be announced later this year.
